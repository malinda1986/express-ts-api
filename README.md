## API service.

### How to run
- cd in to server folder
- run yarn
- Make sure 9006 port is available [you can change the port in src/config/index.ts] .
- run ```make start```
- unit test ```yarn test:unit```
- API doc ```http://localhost:9006/v1/api-doc```
